# mesh 좌표축 config
mesh의 원점이 ee_link에 부착되도록 설정
+z방향 : 통신선
+x방향 : gripper부착면

radius : 41mm
depth : 25.40mm

-> visual mesh는 이걸로 하되, collision은 cylinder로  -> urdf의 오류가 있어 일단 visual mesh와 동일하게 설정 (hdr35_20_ati_dg5f_l.urdf에서)

flange : 5.1mm

code example
<link name="axia_80_ft_sensor">
  <visual>
    <origin xyz="0 0 0" rpy="0 0 0"/>
    <geometry>
      <mesh filename="meshfile dir"/>
    </geometry>
  </visual>

  <collision>
    <origin xyz="0.0127 0 0" rpy="0 0 0"/>
    <geometry>
      <cylinder radius="0.041" length="0.0254"/>
    </geometry>
  </collision>
  
  </link>
